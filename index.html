What is a DBR laser?
device overview
The yellow dot = photon bouncing between the two DBR mirrors, building up to lase.
DBR mirrors
Stacks of alternating high/low refractive index layers. Each pair partially reflects light; together they reach R ≥ 99%.
Quantum well (click to explore →)
Ultra-thin GaAs layer (~10 nm). Electrons and holes are confined here and recombine to emit photons.
Laser output
Photons bounce between the two DBR mirrors (the cavity), amplified each pass, until stimulated emission builds up and lases.
Animated photon (hν)
Yellow dot shows a photon emitted from the QW reflecting off each DBR mirror. The flashes at the DBR walls show the reflection events.
Quantum Well — heterostructure & photon emission
Physical cross-section
The GaAs well is sandwiched between wider-bandgap AlGaAs barriers. Electrons and holes are quantum-confined in the well, raising their ground state energies above the band edge.
Band diagram + photon emission
Electrons (blue) occupy the quantized subband E₁c; holes (open circle) occupy E₁v. When they recombine, a photon is emitted with energy hν = E₁c − E₁v. This photon then bounces between the two DBR mirrors, gaining stimulated emission gain on each pass until the round-trip gain exceeds loss → lasing.
How the emitted photon is reflected by the DBR
The photon emitted from the QW hits the DBR stack. Each layer interface partially reflects the wave. Because every layer is λ₀/4n thick (quarter-wave), all partial reflections arrive back in phase — they add constructively, achieving R ≥ 99%.

Formula: how R(λ) is computed
TMM equations — click to expand
Variables — what n_H and n_L mean
n_HHigh refractive index layer (e.g. GaAs, 3.59). Thinner thickness for same phase shift.
n_LLow refractive index layer (e.g. AlAs, 2.94). Optically sparse.
d_Hλ₀ / (4·n_H)  ← quarter-wave thickness
d_Lλ₀ / (4·n_L)
NNumber of H/L pairs. More periods → higher R.
n₀Refractive index of incident medium (air = 1.0)
n_SSubstrate refractive index
Why quarter-wave? d = λ₀/4n gives phase δ = π/2. All partial reflections from each interface add up in phase → constructive interference → R → 100%.
Transfer matrix for one layer M_j
Mj = cos δj (i/nj) sin δj i·nj sin δj cos δj
δj = 2π · nj · dj / λ  ← phase thickness
At quarter-wave: δ = π/2 → cos δ = 0, sin δ = 1
System matrix (N periods)
Mtotal = (MH · ML)N
Multiply 2×2 matrices N times
→ get M₁₁, M₁₂, M₂₁, M₂₂ (complex)
Reflectance R(λ) — the graph
r = n₀M₁₁ + n₀nsM₁₂ − M₂₁ − nsM₂₂ n₀M₁₁ + n₀nsM₁₂ + M₂₁ + nsM₂₂
R(λ) = |r|²  ← plotted in section 3
Repeat for each λ in the sweep. The flat region around λ₀ is the stopband — the high-reflectance window of the DBR mirror.
Closed-form peak R formula (at exactly λ₀)
Rpeak = nS·(nH/nL)2N − n0 nS·(nH/nL)2N + n0 2
Verify: GaAs/AlAs, N=12 → (3.59/2.94)²⁴ × 3.59 ≈ 433 → R = ((433−1)/(433+1))² ≈ 99.1%. Matches the TMM graph exactly.
Stopband width formula (analytical)
Δλ λ0 = 4 π · arcsin ( nH − nL nH + nL )
Depends only on index contrast, not on N. GaAs/AlAs → Δλ ≈ 97 nm. To widen the stopband: use higher-contrast materials (e.g. Si/SiO₂ gives ~400 nm stopband).

Green curve = |E(x)|² standing wave. This is the light intensity at each position inside the mirror. It oscillates because light waves travelling left and right inside the stack interfere with each other — the same way two people shaking a rope from both ends creates a standing wave.
